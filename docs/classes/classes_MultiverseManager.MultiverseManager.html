    Class MultiverseManager

    Clase que representa el gestor del multiverso, encargado de administrar las dimensiones, personajes, especies, inventos, ubicaciones y viajes. Implementa las interfaces IAdd, IRemove, ISearchElements e IModify para realizar las operaciones correspondientes sobre los elementos del multiverso.
