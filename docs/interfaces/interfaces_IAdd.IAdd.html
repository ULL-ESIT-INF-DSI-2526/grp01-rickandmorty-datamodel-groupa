    Interface IAdd

    Interfaz que define los métodos para agregar elementos al multiverso de Rick and Morty. Esta interfaz es implementada por la clase MultiverseManager para proporcionar funcionalidades de adición específicas para personajes, localizaciones, inventos, dimensiones y viajes en el multiverso.

    interface IAdd {
        addCharacter(new_character: Character): void;
        addDimension(new_dimension: Dimension): void;

        addLocation(new_location: Location): void;
        addSpecie(new_specie: Species): void;
